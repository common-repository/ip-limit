== Description ==
IP制限プラグイン - View filtering by IP

内部閲覧専用のIPを登録し、投稿記事単位の閲覧制御を行います。

<strong>## 制限範囲</stong>
<ul><li>投稿一覧</li>
<li>投稿画面の前ページ&後ページ</li>
<li>最近の投稿</li>
<li>コメント一覧</li>
<li>最近のコメント</li></ul>

※RSSで取得できる記事は「外部公開」の記事のみです。コメントは取得できません。
